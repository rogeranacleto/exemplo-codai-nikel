# Nikel

Este é um repositório do sistema Nikel do projeto [Codaí](https://www.plataforma.growdev.com.br/).

This is a repository for the Nikel system of the [Codaí](https://www.plataforma.growdev.com.br/) project.

Nikel é uma plataforma de controle financeiro que oferece as seguintes funcionalidades:

Tela de login e cadastro: permite que novos usuários se registrem e que usuários existentes acessem o sistema:
<img width="1733" height="1091" alt="image" src="https://github.com/user-attachments/assets/54956fbc-c728-42f6-a9cc-d151745f3492" />


Página inicial: o usuário pode adicionar registros de transações de entrada ou saída, informando valor, descrição e data. Também é possível excluir registros já cadastrados:
<img width="1734" height="1088" alt="image" src="https://github.com/user-attachments/assets/b606488b-3dbb-4746-9d9a-e057fbf9b819" />


Página de lançamentos: exibe todos os registros em uma tabela organizada, separando-os por tipo de operação (entrada ou saída):
<img width="1732" height="1088" alt="image" src="https://github.com/user-attachments/assets/3b411af4-5d23-45f5-bdbf-7442c00b4177" />


O sistema também conta com um menu de navegação que permite alternar entre a página inicial e a página de lançamentos, além de realizar logout. Ele também é responsivo e pode ser acessado facilmente em dispositivos móveis.
