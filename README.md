# Nikel

Este é um repositório do sistema Nikel do projeto [Codaí](https://www.plataforma.growdev.com.br/).

This is a repository for the Nikel system of the [Codaí](https://www.plataforma.growdev.com.br/) project.

Nikel é uma plataforma de controle financeiro que conta com tela de login e cadastro de usuários, além de um menu de navegação que permite alterar entre a página inicial e a página de lançamentos de registros, também é possível realizar o logout. O sistema é responsivo e pode ser acessado facilmente em dispositivos móveis. Abaixo está as funcionalidades detalhadas:



Tela de login e cadastro: permite que novos usuários se registrem e que usuários existentes acessem o sistema:
